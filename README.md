<h1>How To Join the XavExplorer Team</h1>
<p>In this page, we are going to teach you how to join a XavexServer. Easy you think, huh? Well, it's harder than you thought. In order to join our servers, do the following steps:</p>
<h2>Step 1: Download Minecraft Java</h2>
<h4>This is the first important thing needed. The base of Minecraft must be installed inorder to launch it.</h4>
<p>1. Download a <b>Minecraft Launcher</b>. It could be official or cracked.</p>
<p>2. Complete the setup in the .exe file.</p>
<p>3. Create an installation that points to Fabric 1.20.1.</p>
<p>3. <a href="https://www.example.com">Download Java 21 by Oracle</a>.  Complete the installation properly.</p>
<h2>Step 2: Sign in to GitHub with the XavexServer team account.
<p>1. Go to <b>github.com</b></p>
<p>2. Log out of all devices.</p>
<p>3. Click <b>Log In</b>.</p>
<p>4. In the username box, type <b>XavexServerMCTeam</b></p>
<p>5. In the password, type <b>XavExplorerMinecraft</b></p>
<p></p>
<h2>Step 3: Setup Tailscale.</h2>
<p>How if you can't join the server?</p>
<p>1. Try opening Tailscale before going back to Minecraft.</p>
<p>2. Install all the mods required. The mods to all servers can be accessed here. For other mods specifc to that server, check in their server GitHub site.</p>
<p>3. Wait 'till the server is launched. Ask XavExplorer, the admin operator of XavexServers, to launch the server.</p>
